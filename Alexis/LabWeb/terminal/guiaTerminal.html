Guia Terminal

Sintaxis

Comandos

echo

Imprime el argumento que le den

man

Regresa una explicacion sobre el argumento

Por ejemplo: man echo

cat

Coloca el contenido del argumento en la terminal

diff

Muestra la diferencia entre dos documentos

Ejemplo: diff texto1 texto2

mv

Mueve un documento

Ejemplo: mv nombreViejo nombreNuevo

cp

Copia un documento

Ejemplo: cp nombreViejo nombreNuevo

Borra un documento

rmdir

Borra un directorio

wc

Cuenta las letras de un documento

|

Agregar una "operacion" a un resultado

Ejemplo: head texto | wc

Lo que sale de head, se corre con wc

head

Muestra el inicio de un documento

tail

Muestra el final de un documento

grap

Buscas algo en un archivo e imprime todo lo que tenga esa palabra

which

Regresa el camino a un archivo dado

curl

Traer o copiar de un URL

open

Abre un directorio

find

Encuenta un documento. La sitaxis es la siguiente

find dondeBuscas -atributo 'nombre'

Ejemplo: find Desktop -name 'LabWeb'

mkdir

crea un directorio

ls

Muestra lo que hay en el directorio presente

Banderas

-l

Long form

-rtl

reverse time long

-a

muesta documentos escondidos

cd

Cambia al directorio que se da como parametro

cd - te lleva al ultimo directorio en el cual estuviste

touch

Crea un archivo vacio

less

Muestra lo que hay en un archivo pero con interactividad

Se puede utilizar shift G para moverse, tambien puedes buscar con /aBuscar

q

Te regresa a la terminal

pwd

Ver donde estas

>

Operador redireccional

Ejemplo: "Texto" > texto.txt

>>

Operador de adjuntar

!

Corre el utlimo comando que empieza con lo que pongas como parametro

!!

Core el ultimo comando

.

Se refiere al directorio seleccionado

..

Se refiere al siguiente directorio con mayor herarquia

&&

Correr mas de un comando a la vez. Solo se ejecutan los demas si se ejecutaron los anteriores

sudo

Entrar como "super usuario" para hacer cosas que requieran permiso

Ejemplo: sudo touch alexis

Control + C

Cancela la accion que se este realizando

Control + A

Regresa al principio de la linea

Control + E

Regresa al final de la linea

Control + U

Borra la linea

Control + L

Borra la terminal

Control + R

Correr lo ultimo que empieza con...

Alt + Click

Te coloca en donde clickeaste para poder modificar
