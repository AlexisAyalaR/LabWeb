Sintaxis

Para poder estilizar un elemento se tiene que seguir una sitaxis especifica. Esta se puede ver a contiunacion:

Selectors

A continuacion se presentan toda las formas de seleccionar elementos para agregarles estilo:

Seleccionar todo

*{
propiedades: valores;
}

Nombre del elemento

div{
propiedades: valores;
}

Por clase

.nombreClase{
propiedades: valores;
}

Por ID

#nombreID{
propiedades: valores;
}

Modelo de Caja

En el modelo de caja existen cuatro elementos:

Contenido

Es lo que esta dentro de la caja

Padding

El espacio entre el contenido y el borde o las orillas

Border

El borde o las orillas de la caja

Margin

Es el espacio entre la caja y lo demas

Propiedades

Algunas de las propiedades mas coumunes son:

Texto

Propiedades para estilizar texto:

font-family

Cambia la familia

font-style

Cambia el tipo

font-size

Cambia el tamano

color

Cambia el color

font-weight

Cambia que tan ancha es la letra

Posicion

Propiedades para cambiar la posicion de algunos elementos:

position

Coloca elementos en la pantalla

Algunos de los valores que puede tomar son:

absolute

La posicion literal

fixed

La posicion conforme a otras cosas

relative

La posicion conforme al documento

float

Hacia donde esta cargado el "peso"

display

Especifica como esta posicionado el elemento en el flujo del documento

z-index

La "altura" del elemento

overflow

Especifica lo que sucede fuera del contenedor

Otras proiedades comunmente utilizadas:

background

Espeficica el tipo de fondo que se requiere

cursor

Se especifica el cursor

width

Se especifica la anchura

height

Se especifica la altura

Unidades

Existen muchos tipos distintos de unidades pero los mas comunes son:

px

Pixeles, 1px = 0.02645833 cm

%

Porcentaje de la parte visual de la pagina

em

Relativo al tamano de la letra del elemento

vmin

Relativo al 1% de la dimension mas pequena de lo que se ve

vmax

Relativo al 1% de la dimension mas grande de lo que se ve

Media

Se utiliza para modificar el documento conforme a un tamano en especifico

@media only screen and (max-width: X px) {

propiedades: valores; }
