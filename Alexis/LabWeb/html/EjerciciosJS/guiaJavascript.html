Variables

Javascript es un lenguaje de programacion orientado a objetos. Por tanto se pueden declarar variables, pero a diferencia de otros lenguajes, no se tiene que especificar el tipo de variable:

var nombreVariable = valor;

Objetos

Tambien se pueden crear objetos en forma de variables donde se pueden especificar todas las caracteristicas necesarias. El formato que se utiliza se conoce como JSON:

var nombreObjeto = {
propiedad1: "Valor1",
propiedad2: "Valor2",
};

Funciones

Tambien se pueden crear funciones, solamente no se especifica el tipo del dato que se regresa. La sintaxis es la siguiente:

function nombreFuncion( parametro1, parametro2 ) {

codigo;

codigo;

codigo;

return variable;

}

No tiene porque regresarse una variable, pero es algo comun.

Ciclos y Declaraciones

Al igual que con otros lenguajes orientados a objetos, existen ciclos. La sintaxis que se usa se presenta a continuacion:

if-else

if( condicion ) {

Si se cumple, corre este codigo

}

else{

Si no se cumple, corre este

}

for

for (var i = tamano ; i <= variable; i++) {

Se ejecuta este codigo

};

while

while( condicion ) {

codigo a ejecutar

}

Metodos importantes de JS

Document

Existen metodos que modifican o permiten conocer ciertas partes del documento:

document.getElementById( "id" )

Permite accesar a un elemento del documento a partir de su id

document.createElement( "tipo" )

Permite crear un nuevo elemento

document.createTextNode( "texto" )

Crea un texto para pegarselo a un objeto

Modificar elementos

elemento.appendChild

Agrega un elemento a otro elemento como su "hijo"

elemento.value

Muestra el valor que tiene el elemento

element.setAttribute("atributo", "nombre")

Permite agregar o cambiar un atributo a un elemento

elemento.innerHTML

Escribe sobre el elemento en el documento

alert( "Mensaje" )

Despliega un mensaje

Estilo

elemento.style.nombreMetodoCSS = "valor"

Al tener un elemento tambien se puede modificar el estilo que este tiene con los metodos de CSS. La unica diferencia es que si un metodo tiene un guion, se tendra que escribir seguido y con mayuscula. Por ejemplo margin-top seria marginTop

Eventos

Para poder crear un evento, primero se tiene que poder tener acceso al elemento con el que se disparara. Al ya tener el elemento, se agrega el evento con el siguiente metodo:

elemento.addEventListener( "formaDisparo" , function() {
funcionAEjecutar( parametro1 , parametro2 );
});

Tambien se puede llamar el metodo desde HTML agregando la forma en la que se disparara (por ejemplo "onclick") como atributo:

onclick = "funcion()"
