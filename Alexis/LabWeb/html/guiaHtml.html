DOCTYPE

Define el tipo de documento. Siempre que utilizemos HTML5 se escribira:

!DOCTYPE html

Elementos Principales

html

Es el elemento base de html, la ruta. Todo tiene que estar dentro de esta etiqueta. Puede tener el atributo lang que especifica el lenguaje

html lang="en-US"

head

Contiene toda la informacion meta del documento

title

Especifica el titulo del documento

body

Define toda la parte visible del documento, todo el cuerpo

p

Crea un parrafo

br

Se utiliza para saltar lineas

pre

Es un parrafo que preserva espacio y saltos de linea

Links e Imagenes

a

Define un link. Se tiene que especificar un atributo href tipo String con la URL como parametro

Como este que lleva a W3Schools

a href="link"

img

Define una imagen. Se tiene que especificarlos atributos: src (source), alt (alternative text), width y height, tambien acepta gifs. Todos tipo String

img src="link" alt="texto" width="tamaño" height="tamaño"

map

Define una imagen con areas clickeables

area

Define en area para un image-map
