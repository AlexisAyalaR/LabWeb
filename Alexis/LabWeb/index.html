Guia para HTML

DOCTYPE

Define el tipo de documento. Siempre que utilizemos HTML5 se escribira:

!DOCTYPE html

Elementos principales

html

Es el elemento base de html. Todo tiene que estar dentro de esta etiqueta. Puede tener el atributo lang que especifica el lenguaje

html lang="en-US"

head

Contiene toda la informacion meta del documento

title

Especifica el titulo del documento

body

Define toda la parte visible del documento

p

Crea un parrafo

br

Se utiliza para saltar lineas

h's

Define un encabezado. Mientras mas pequeño sea el numero, mas grande es la letra [1,6].

h1

h2

h3

h4

h5
h6

Links e Imagenes

a

Define un link. Se tiene que especificar un atributo href tipo String con la URL como parametro

Como este que lleva a W3Schools

a href="link"

img

Define una imagen. Se tiene que especificarlos atributos: src (source), alt (alternative text), width y height. Todos tipo String

img src="link" alt="texto" width="tamaño" height="tamaño"
